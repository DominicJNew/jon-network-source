BlopBlog: Blog with Ease

BlopBlog (GitHub) is a software tool that lets you blog easier. Simply blop! all your ideas in, then when you are ready to write you will have all the ideas you need to make your blog posts awesome!


DevChat: Chat for Devs

DevChat is a free immutable chat room for developers to easily colaborate across the web with zero setup required.


The Programming 4um

The Programming 4um is a forum to answer all your why? code questions, helping you learn better code rather than just getting the answer.


Comical IDE

ComicalIDE is a typical HTML, CSS and JavaScript IDE, except that it is in Comic Sans rather than monospace, making you laugh as you type because you wonder who came up with such a stupid font.
